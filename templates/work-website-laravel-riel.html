002 : Webshop development with Laravel framework.

The company's website and online store were built using the Laravel framework in PHP. The backend is closely integrated with an MsSQL database that contains the products of the webshop, ensuring continuous synchronization between the website and the backend. A separate MySql database handles customer data, accompanied by a user interface that manages purchase history and warranty processes. An online payment solution was also implemented during development. The entire coding process followed the Agile (Scrum) methodology with full SDLC application. The website continuously expanded with new features, such as a product selector and an instructional video page.
