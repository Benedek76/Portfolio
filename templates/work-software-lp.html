004 : LastPass trial feature modul development.

The main task was to break down and refactor the legacy "spaghetti" code into modules. During the project, we followed the Agile (Scrum) methodology in a team typically consisting of 8-10 members. We participated in the project as contractors. Our responsibility was to accurately translate the business logic into code using PHP. From the project onboarding, we went through several sprints and epics, leading to the handover of the final code to the client after the final testing, followed by the deployment. The main issue was the incorrect handling of the trial period, which arose from the different timings of ordering the promotional period.
