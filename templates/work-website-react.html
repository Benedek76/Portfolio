006 : React test development, Pizza menu and Profile card.

This React project is a simple pizza webshop menu (and Profile card also) designed to help with learning the basics of React, including setting up the development environment and creating reusable components. The application displays a list of pizzas along with their details, such as ingredients, price, and availability, using React components. The project involves dynamic rendering with JavaScript, demonstrating how to build interactive elements. It also provides a foundation for further expansion with backend logic and database integration if needed.
Key technologies used: React for building UI components JavaScript for dynamic interactivity HTML5 and CSS for structure and styling
