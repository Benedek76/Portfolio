008 : Grace Website.

The website created for the performing artist aims to visually represent the musical world that is characteristic of them. Based on this, I built the project to be sufficiently diverse, clean, and functionally well-usable for fans. The website is built on WordPress, featuring several custom designs, integrated with various social media platforms. To play music tracks, I implemented Spotify, allowing users to instantly play songs. Video playback is achieved through YouTube embedding. Additionally, the website is connected to all the artist’s other social media platforms, such as Apple Music, Facebook, and TikTok accounts. Furthermore, a Shopify webshop was also created, allowing buyers to directly link back to the artist’s website.
