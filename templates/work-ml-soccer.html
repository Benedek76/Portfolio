004 : Soccer Data Insights - Exploring Player Wage and Value Relations

This Jupyter Notebook is a Python-based machine learning project that focuses on analyzing and visualizing soccer data.
The project centered around analyzing soccer player statistics, such as wages and player values. It makes use of data visualization techniques to present the relationship between these variables, possibly aiming to discover patterns or correlations.
Technologies and Libraries Used:
1.Pandas: The project utilizes Pandas for data manipulation and exploration, specifically to load, clean, and organize data for further analysis.
2. Seaborn: This library is used for creating scatter plots to display relationships between features like player wages and values. - Example: A scatter plot is created to show the relationship between player wages and values.
3. Bokeh: Bokeh is used for interactive visualizations. It allows the user to create an interactive plot where hovering over data points reveals additional information like player names, wages, and values. - Example: Bokeh's `figure` and `HoverTool` are used to create an interactive scatter plot, enhancing the data exploration experience.
4. Numpy: As commonly used in data analysis to handle numerical operations.
5. Scikit-learn: For visualization to prediction.
6. Matplotlib: This library is used alongside Seaborn for generating standard visualizations.
Visualizations: - Scatter plots using Seaborn and Bokeh are the main types of visualizations, offering both static and interactive views of the data. - Seaborn creates a more static visualization, while Bokeh adds interactivity, allowing users to hover over points and gain detailed insights, such as the player's name and wage information.
Summary: The project primarily revolves around analyzing and visualizing soccer player data using advanced Python libraries, with a focus on interactive plotting and user-friendly visual analytics.
