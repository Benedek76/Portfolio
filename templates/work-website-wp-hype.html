007 : Hype Energy Drink Company Website development.

The purpose of this project was to develop a modern website that aligns with the refreshed (2023) online branding of Hype Energy Drink. The goal was to present the newly introduced sports nutrition products in a high-quality manner. A requirement of the project was the use of the WordPress framework, but I also had the opportunity to implement numerous custom code and scripts using PHP and JavaScript. These included:
A custom background linear gradient that is triggered upon user scrolling, utilizing JavaScript.
Drop-down fields on product pages to display product-specific information, such as the ingredients table and ingredient list.
Creation of tailored HTML tables for each product.
A custom footer developed using HTML5, CSS3, and PHP.
A custom responsive menu design. The main WordPress plugins used were Instagram Feeding for the sincronize with the Instagram and Motion Page for animations.
Multilanguage feature for several language.
Additionally, I implemented a custom loop grid for News Posts.
I carried out full SEO optimization with the help of an SEO plugin.
