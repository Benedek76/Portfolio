001 : Aron Benedek Ladanyi Portfolio website

A Python-based portfolio website built with the Flask mini framework, uploaded from GitHub and hosted on pythonanywhere.com. The application features a custom-coded Python server (server.py) that manages routes and various functions, such as page redirection, form submission, and writing messages to a CSV file. Flask modules like render_template, request, and redirect were utilized, along with the CSV module. The frontend was created by heavily recoding a pre-existing template and adding/modifying JavaScript functions. The site has a clean, modern design, including a custom home page image (created in CorelDraw), ensuring simple navigation. Visitors can send messages via the contact form.
